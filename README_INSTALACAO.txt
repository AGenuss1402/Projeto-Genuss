                    GUIA DE INSTALAÇÃO

Bem-vindo ao Genuss Automação!

Este sistema oferece uma solução completa para ponto de venda com:

✓ Emissão de NFCe (Nota Fiscal de Consumidor Eletrônica)
✓ Controle completo de estoque
✓ Gestão de vendas e clientes
✓ Relatórios gerenciais detalhados
✓ Interface moderna e intuitiva
✓ Suporte a impressoras térmicas
✓ Backup automático de dados

=================================================================
                        PRÉ-REQUISITOS
=================================================================

Antes de prosseguir com a instalação, certifique-se de que seu
sistema atende aos seguintes requisitos:

• Sistema Operacional: Windows 10 ou superior (64-bit)
• Memória RAM: Mínimo 4GB (Recomendado 8GB)
• Espaço em disco: Mínimo 2GB livres
• Microsoft .NET Framework 4.8 ou superior
• Microsoft Access Database Engine 2016 Redistributable
• Conexão com a internet (para emissão de NFCe)

=================================================================
                    COMPONENTES INCLUSOS
=================================================================

Este instalador inclui:

• Aplicação principal Genuss Automação
• Banco de dados SQLite integrado
• Bibliotecas de terceiros necessárias:
  - ACBrLib (Automação Comercial Brasil)
  - ZeusNFe (Emissão de NFCe)
  - Bunifu UI Framework
  - QRCoder (Geração de QR Code)
  - Aspose.PDF (Manipulação de PDF)
  - Newtonsoft.Json (Processamento JSON)

=================================================================
                      APÓS A INSTALAÇÃO
=================================================================

1. Execute o sistema como Administrador na primeira vez
2. Configure os dados da empresa em Configurações > Empresa
3. Configure o certificado digital em Configurações > NFCe
4. Teste a conexão com a SEFAZ
5. Configure as impressoras em Configurações > Impressoras
6. Cadastre os produtos iniciais
7. Realize uma venda de teste
